# Advanced JVM and String Internals Interview Questions

This document contains advanced interview questions for senior Java
developers, SDETs, automation architects, and JVM enthusiasts.

Topics covered: - String Constant Pool - Heap, Stack, Metaspace - Class
Loading - Garbage Collection - String Deduplication - StringBuilder vs
StringBuffer - equals() and hashCode() - Escape Analysis - JVM
Optimizations
